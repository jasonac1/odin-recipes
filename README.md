# Foundations: Recipes

This is my take on the Recipes project of The Odin Project. The project aims to help practice building a simple website with text, images, and links using HTML.

Project structure:
```bash
. (odin-recipes)
├── images
│   ├── chicken-arroz-caldo.jpg
│   ├── chocolate-chip-cookies.jpg
│   └── steamed-eggs.jpg
├── index.html
├── README.md
├── recipes
│   ├── chicken-arroz-caldo.html
│   ├── chocolate-chip-cookies.html
│   └── steamed-eggs.html
└── styles.css

```
